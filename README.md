## What I'm Building

**[Ayvu-Talian](https://github.com/jorgeguberte/ayvu-talian-base)** — Decoder-only transformer trained from scratch for the Talian dialect (Brazilian Venetian). Because low-resource languages deserve to exist in AI too.
